Detailed Description

Class FileReader is responsible for reading the *.txt file.

The class is used to read the dimensions input file that contains the dimensions of the Sculpture in x, y and z. And you receive which type of figure should be drawn or deleted from the Sculpture, in each line of the * .txt file
